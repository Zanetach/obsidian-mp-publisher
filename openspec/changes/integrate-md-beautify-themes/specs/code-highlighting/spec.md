## 新增需求

### 需求:代码语法高亮
系统必须使用 highlight.js 为代码块提供语法高亮，支持的语言必须包括 JavaScript、TypeScript、Python、Java、C++、Go、Rust、HTML、CSS、JSON、YAML、Markdown 等常见语言。

#### 场景:JavaScript 高亮
- **当** 预览内容包含 ```javascript 代码块
- **那么** 系统必须应用 JavaScript 语法高亮样式

#### 场景:Python 高亮
- **当** 预览内容包含 ```python 代码块
- **那么** 系统必须应用 Python 语法高亮样式

### 需求:代码主题适配
系统必须根据当前主题选择合适的代码高亮样式，确保与整体视觉风格一致。

#### 场景:代码高亮主题匹配
- **当** 用户切换主题
- **那么** 代码高亮样式必须与新主题匹配

### 需求:行号显示
系统可以选择为代码块显示行号。

#### 场景:显示行号
- **当** 用户启用行号显示选项
- **那么** 代码块必须显示行号

### 需求:复制代码
复制代码时，系统必须只复制代码内容，不包含行号和语法高亮标记。

#### 场景:复制纯代码
- **当** 用户点击复制按钮复制代码块
- **那么** 剪贴板必须只包含纯代码文本
