## 新增需求

### 需求:行内公式渲染
系统必须支持 KaTeX 行内公式渲染，使用 `$...$` 语法。

#### 场景:渲染行内公式
- **当** 预览内容包含 `$E=mc^2$` 语法
- **那么** 系统必须将其渲染为 KaTeX 格式的行内公式

#### 场景:渲染复杂行内公式
- **当** 预览内容包含 `$\int_{a}^{b} f(x) dx$` 语法
- **那么** 系统必须将其渲染为 KaTeX 格式的积分公式

### 需求:块级公式渲染
系统必须支持 KaTeX 块级公式渲染，使用 `$$...$$` 语法。

#### 场景:渲染块级公式
- **当** 预览内容包含 `$$\int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$` 语法
- **那么** 系统必须将其渲染为 KaTeX 格式的居中块级公式

### 需求:公式字体样式
系统必须为 KaTeX 渲染结果应用与主题一致的字体样式。

#### 场景:公式字体适配
- **当** 渲染 KaTeX 公式
- **那么** 公式必须使用与主题匹配的字体

### 需求:微信复制兼容
复制包含 KaTeX 公式的内容到微信时，系统必须将公式转换为图片或保留可读格式。

#### 场景:复制时保留公式
- **当** 用户复制包含 KaTeX 公式的内容
- **那么** 复制的内容必须包含可显示的公式或保留 LaTeX 源码
